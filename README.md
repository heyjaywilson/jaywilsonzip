## Local Development

To get started:

  ```bash
  # Install dependencies
  npm install

  # Start development server
  npm run dev

  # Build for production
  npm run build
  ```

The dev server will watch for changes and automatically rebuild CSS and site content. Access the site at `localhost:8080`.
